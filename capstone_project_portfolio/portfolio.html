Project Description

Gradslist is a modern online marketplace platform similar to OfferUp and Facebook Marketplace. It allows users to post items, browse listings, search across categories, and buy/sell products in a streamlined and user-friendly environment. Our goal was to create a fast, clean, and reliable marketplace powered by real-time Firebase data, dynamic filters, and an optimized search experience.

My Role & Contributions

As the project manager and lead front-end developer, I directed the technical vision of the project and implemented core functionality used across the application:

  • Project Management & Architecture
    • Led all sprints, task delegation, and weekly planning
    • Set up React environment, Vite configuration, and the entire folder structure
    • Managed the GitHub repo, branches, pull requests, and code reviews
    • Ensured team followed consistent coding standards and workflow
  • Front-End Development
    • Built global Header, Footer, layout system, and UI structure
    • Created the category-based subheader with clickable filters for every listing type
    • Rebuilt homepage to display fully dynamic Firestore listings instead of static items
  • Search System (My Largest Feature)
    • Developed the full search bar using Firestore queries
    • Implemented smart suggestions and error messaging
    • Added fallback logic when filters conflict or missing data occurs
    • Optimized queries for speed and reliability
  • Backend & Firestore Contributions
    • Assisted in designing Firestore collections & fields
    • Connected homepage listings to Firestore for real-time display
    • Debugged Firestore read issues and improved query error handling
  • Debugging & Stability
    • Resolved Vite “import-analysis” errors and broken routing imports
    • Cleaned component structure and removed unused code
