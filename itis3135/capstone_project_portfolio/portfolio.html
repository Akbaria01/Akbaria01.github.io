Project Description

Gradslist is a modern online marketplace platform similar to OfferUp and Facebook Marketplace. It allows users to post items, browse listings, leave reviews on item and Seller profiles, search across categories, and buy/sell products in a streamlined and user-friendly environment. Our goal was to create a fast, clean, and reliable marketplace powered by real-time Firebase data, dynamic filters, and an optimized search experience.
